## 집밥 레시피(우리url첨부)


<소개> **우리집만의 레시피를 공유하는 플랫폼**

우리가 집에서 해먹는 음식들 레시피를 다른 사람들과 공유해보아요  
'집밥레시피 소개 영상'(소개유튜브 url첨부)
   
## 프로젝트 기간 : 2022.09.02.금~ 2022.09.08.목  
   
## 주요기능
### 1. 회원 시스템 - 회원가입, 로그인 (이메일, 닉네임)
### 2. 게시글 - 레시피 조회 및 업로드(생성, 수정, 삭제)
### 3. 게시글 좋아요 - 좋아요 갯수 확인, 취소 가능
### 4. 댓글 - 게시글 별 댓글 생성,삭제
